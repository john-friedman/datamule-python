Welcome to datamule’s documentation!

A Python package to work with SEC submissions at scale. Integrated with datamule’s APIs and datasets.

Features

Core Functionality

  • Fast SEC Downloads: Utilize the Downloader or Premium Downloader to download SEC filings.

  • Real-time Monitoring: Keep track of new EDGAR submissions using Monitor.

  • Interact with SEC Submissions: Access, parse, and interact with SEC submissions using Portfolio, Submission, and Document.

Available Datasets

Access SEC-related datasets:

  • Company Metadata (including SIC Codes)

  • Company Former Names

  • Company Tickers

  • SEC Submission Types Glossary

  • XBRL Descriptions

MuleBot Integration

Note

I built MuleBot in a day to learn how tool-calling chatbots with artifacts work. I’m planning to make it useful in the future. Still fun to play with!
